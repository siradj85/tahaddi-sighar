# تحدي الصغار 🌟

لعبة أسئلة تعليمية وممتعة للأطفال باللغة العربية (على نمط "من سيربح المليون" لكن للصغار).
مبنية بـ **Kotlin + Jetpack Compose** لنظام أندرويد، مع إعلانات مكافئة (AdMob) للربح.

## المميزات
- أسئلة متنوعة: حيوانات، ألوان، أرقام، حروف، معلومات عامة، آداب.
- واجهة عربية ملوّنة ومناسبة للأطفال (من اليمين لليسار).
- نظام نقاط + حفظ أفضل نتيجة.
- مساعدة "احذف إجابتين" مقابل مشاهدة إعلان مكافئ (مصدر ربح).
- مضبوط على سياسة Google Play "مصمّم للعائلات".

---

## كيف أحصل على ملف التطبيق للرفع؟ (بدون تثبيت أي برنامج)

1. ارفع هذا المجلد إلى مستودع GitHub جديد (سيتولى المساعد ذلك).
2. سيعمل البناء تلقائياً عبر **GitHub Actions** (تبويب Actions في المستودع).
3. بعد دقائق، افتح آخر تشغيل (Run) ومن قسم **Artifacts** ستجد:
   - `aab-للنشر-على-جوجل-بلاي` → ملف **AAB** ترفعه في Google Play Console.
   - `apk-تجريبي-للتثبيت` → ملف **APK** ثبّته على هاتفك لتجربة اللعبة فوراً.
   - `مفتاح-التوقيع-احتفظ-به` → **مفتاح التوقيع** (يظهر أول مرة فقط — احفظه جيداً، تحتاجه لكل تحديث مستقبلي).

> كلمة مرور المفتاح الجديد تظهر في سجل التشغيل (Logs) ضمن خطوة "تعليمات المفتاح الجديد".

---

## قبل النشر النهائي (مهم)
- استبدل معرّفات AdMob الاختبارية بمعرّفاتك الحقيقية:
  - معرّف التطبيق في `app/src/main/AndroidManifest.xml`
  - معرّف الإعلان في `app/src/main/java/.../AdManager.kt` (`REWARDED_AD_UNIT_ID`)
- في Google Play Console: املأ قسم **الجمهور المستهدف والمحتوى** واختر "للأطفال"، واملأ استبيان الإعلانات.

## كيف أضيف أسئلة جديدة؟
افتح ملف `app/src/main/java/com/saidcharoun/tahaddisighar/QuestionBank.kt` وأضف سطراً جديداً على نفس النمط.
